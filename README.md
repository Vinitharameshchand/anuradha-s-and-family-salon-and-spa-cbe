# Anuradha's Salon Prototype

A small Vite + React prototype demonstrating the Navbar, Hero, and Services sections.

Quick start:

```bash
cd prototype
npm install
npm run dev
```

Design notes:
- Colors and typography follow the provided brand tokens.
- Buttons, card hover, and gradient behavior included.
Files:
- `src/components` — React components
- `src/styles.css` — core styles
